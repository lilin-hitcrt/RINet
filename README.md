# RINet

### Environment
#### Conda
```
conda create -n rinet python=3.7
conda activate rinet
conda install pytorch torchvision torchaudio cudatoolkit=11.1 -c pytorch -c nvidia
conda install tqdm scikit-learn matplotlib tensorboard
```
